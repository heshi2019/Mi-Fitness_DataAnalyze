## MiFitness_hlth_center_fitness_data.csv（传感器底层）

➡️ **所有原始的时间序列数据（steps、心率、calories、headset、压力、血氧等）**
 这是采样级数据。例如（心率、步数、卡路里、血氧、压力等）累计等。

## 📌 字段含义

| 字段           | 说明                                         |
| -------------- | -------------------------------------------- |
| **Uid**        | 用户 ID                                      |
| **Sid**        | 数据源 ID（不同手表会对应不同 SID）          |
| **Key**        | 数据类型（hr、steps、calories、pa、spo2 等） |
| **Time**       | 时间戳（秒级）                               |
| **Value**      | JSON 内容（因 Key 不同结构不同）             |
| **UpdateTime** | 更新时间戳                                   |

### 📌Key值含义

| key值               | 含义                                                         |
| ------------------- | ------------------------------------------------------------ |
| headset             | 耳机相关数据（推测为耳机连接 / 使用状态记录）                |
| steps               | 步数数据（包含步数、距离、卡路里消耗）                       |
| calories            | 卡路里消耗数据（单独记录的卡路里数值）                       |
| dynamic             | 动态运动数据（综合记录单次运动的步数、距离、卡路里、运动类型等） |
| valid_stand         | 有效站立数据（记录站立时间段）                               |
| heart_rate          | 心率数据（实时 / 定时心率记录）                              |
| stress              | 压力值数据（记录身体压力状态评分）                           |
| pai                 | PAI 健康指数数据（个人活动智能指数，含不同强度区间 PAI 值）  |
| watch_daytime_sleep | 手表记录的日间睡眠数据（含睡眠片段、时长等）                 |
| sleep               | 睡眠数据（综合睡眠记录，含深浅睡眠、清醒次数等详细信息）     |
| watch_night_sleep   | 手表记录的夜间睡眠数据（含深 / 浅 / REM 睡眠、清醒次数等详细分段） |
| resting_heart_rate  | 静息心率数据（休息状态下的基础心率）                         |
| intensity           | 运动强度数据（记录运动强度相关标记）                         |
| weight              | 体重数据（含 BMI、体重数值）                                 |
| single_heart_rate   | 单次心率数据（单独记录的某一次心率测量值）                   |
| noise               | 噪音数据（记录环境噪音分贝值）                               |

### 补充说明

1. **时间戳说明**：所有`time`/`start_time`/`end_time`等字段均为 Unix 时间戳（秒级），可通过在线工具转换为常规日期时间（如`1765283262`转换为`2025-08-09 10:14:22`）。

2. **睡眠状态编码**：结合行业常见标准推测：`2=深睡`、`3=浅睡`、`4=REM睡眠`、`5=清醒`、`6=其他睡眠相关状态`。

   睡眠字段表示含义还有待验证

3. **数据来源**：`Sid`字段区分数据来源，`hlth.gen_`开头为小米健康通用数据源，`huami.`开头为华米设备数据源，`xiaomiwear_app_manually`为小米穿戴 APP 手动录入数据。
